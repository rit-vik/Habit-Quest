## Habit Quest

Habit Quest is a habit-tracking web application designed to help users stay consistent with their daily goals. It provides a simple to-do interface along with a reward-based system that motivates users to complete tasks and build better habits over time.

## Overview

The goal of this project is to combine productivity with gamification. Users can add and manage their daily tasks, track their progress, and earn rewards for completing tasks, making the experience more engaging than a traditional to-do list.

## Features

- Add, update, and delete daily tasks  
- Track progress through a structured to-do list  
- Reward system based on task completion  
- Clean and responsive user interface  
- Real-time interaction with tasks
